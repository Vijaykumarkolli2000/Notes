Once done with msg from producer and consumer -
 
To consume a Kafka topic and display both the key and value using the Kafka-console-consumer in new cmd command, you can use the –property option to set the print.key and print.value properties to true.
use below command- 
 
 
 .\bin\windows\Kafka-console-consumer.bat --bootstrap-server localhost:9092 --topic my-topic --formatter kafka.tools.DefaultMessageFormatter --property print.timestamp=true --property print.key=true –property print.value=true --from-beginning
